Firmware ESP32 + ADAU1701 — Web Installer

Conecta tu ESP32 por USB y pulsa el botón para instalar el firmware.

Requiere Google Chrome o Microsoft Edge en escritorio.

Tu navegador no soporta Web Serial.
Usa Chrome o Edge en escritorio. No se permite el acceso al puerto serie en este contexto.

Instrucciones

  1. Conecta el ESP32 a tu ordenador con un cable USB.
  2. Pulsa "Instalar Firmware" y selecciona el puerto serie del ESP32.
  3. Espera a que termine la instalación (~1 minuto).
  4. El ESP32 se reiniciará automáticamente con el nuevo firmware.
  5. Conéctate a la red WiFi del dispositivo y accede a radiocassete.local.
Nota: La primera vez puede ser necesario mantener pulsado el botón BOOT del ESP32 mientras se inicia la conexión. Si tienes problemas, asegúrate de tener instalados los drivers CP210x o los drivers CH340 según tu placa.
